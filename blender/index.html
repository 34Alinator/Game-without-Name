Substraktive Box

  1. Auf die Box ,von der substrahiert werden soll, drücken
  2. Auf den Schraubenschlüssel gehen(rechts unten)
  3. Modifier hinzufügen
  4. Boolean auswählen
  5. Modus "Difference
  6. Auf die kleine Pipette neben objekt drücken damit dann auf die Box mit der man substrahieren will drücken
  7. Mit "H" oder oben rechts die Substraktive Box ausblenden

Objekt halbieren

  1. Box Kopieren und an gleiche stelle einfügen
  2. Eine Box erstellen die die hälfte abdeckt
  3. Eine Box zu einer Substraktieven Box machen
  4. Andere Box zu einer Intersect Box machen
  5. Booleans(Substraktieve und Intersect) zuweisen
  6. Substrektieve Box ausblenden
